# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and abusive behavior are not
tolerated in issues, pull requests, or any other project space.

Report unacceptable behavior to the maintainer at **rashid.azarang.eg@gmail.com**. Reports are handled
confidentially and in good faith. The full text of the pledge, standards, scope, and enforcement
guidelines is at the link above.
